# E-Client
E-Client is a mod menu for the popular VR game Gorilla Tag. This menu is constantly being update so enjoy

---

# Installation

## Method 1.

1. Download the file
2. put the menu in .../Gorilla Tag/BepInEx/plugins
3. Enjoy the menu

## Method 2.

1. Download the installer
2. Enjoy the menu
